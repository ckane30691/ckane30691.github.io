Frogger Game

This is a clone of the classic Frogger game written in Javascript using object oriented principles. The goal is to get the highest score possible without getting hit by bugs.

Ayyy Lmao Bot

This is a Reddit Bot that I coded up for fun in an afternoon. It's written in Python and scans the comment section for variations on the word ayyy. If it finds one it responds with lmao. It then adds that comment to an array which it checks against so that it doesn't keep responding to the same comment over and over each time it scans.

Purchase Order Form Browser Extension

This is a B2B Google Chrome Extension I developed for a client that adds Purchase Order functionality to the popular Basecamp collaboration website. The extension itself injects javascript into the Basecamp project overview page which adds a link to a purchase order page, hosted by the client, allowing their customers (who have the extension installed) to order from them.
